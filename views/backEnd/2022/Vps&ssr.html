    搭建VPS服务器，并开启SSR+BBR加速

    Samer 3/8/2022 linux梯子

    # 1.准备VPS服务器

    # 1.选择一个合适套餐的VPS租赁即可，例如搬瓦工，hostwinds，等等，此部分略过

    # 2.安装CentOS 7系统

    在对应VPS服务商后台管理界面安装CentOS 7系统即可，选择这个版本是因为其他版本或多或少都对后续安装和使用脚本有影响

    # 2.脚本安装及使用

    # 0.准备工作

    #创建个人文件夹
mkdir /samer
cd /samer
#安装必要文件
yum install -y wget
    1
    2
    3
    4
    5

    # 1.建议首先安装BBR/BBR魔改版加速脚本（解释在后文说明处）

    wget -N --no-check-certificate "https://raw.githubusercontent.com/chiakge/Linux-NetSpeed/master/tcp.sh"
chmod +x tcp.sh
./tcp.sh
    1
    2
    3

    1.选择安装

    2.完成后根据提示进行重启服务器

    3.重新连接后再次运行tcp脚本，开启BBR/BBR魔改版/暴力魔改版加速

    4.加速完毕后，输入以下命令查看

    # 查看内核
uname -r
    1
    2

    如果返回值含有4.x以上版本，就表示ok

    #查看加速版本
sysctl net.ipv4.tcp_available_congestion_control
    1
    2

    sysctl net.ipv4.tcp_congestion_control
    1

    如果返回值中含有bbr表示普通BBR加速，含有tsunami表示魔改版加速，含有nanqinlang表示为暴力魔改版加速

    sysctl net.core.default_qdisc
    1

    返回值一般为：net.core.default_qdisc = fq

    到此差不多认为BBR安装并启动成功

    # 2.安装SSR脚本

    wget -N --no-check-certificate https://raw.githubusercontent.com/luvvien/ssr-install-shellscript/master/ssr.sh
chmod +x ssr.sh
bash ssr.sh
    1
    2
    3

    选择安装 ShadowsocksR 根据提示一步一步进行安装，等待安装完毕即可

    # 3.安装客户端

    各版本链接：https://www.magento2u.com/archives/1923 (opens new window)

    IOS客户端需要在外服App Store下载Potatso Lite 或者 Shadowrock等SSR软件，后者付费，功能更强大

    # 4.KCPTUN加速?

    有兴趣可以搜索相关文章，加速效果更强大

    这里贴一个文章链接参考：https://ssr.tools/588 (opens new window)

    # 5.说明

    # 1.解释脚本安装先后顺序

    如果先安装了SSR脚本再安装BBR加速，会发现客户端连不上SSR了，这时候就需要再次执行ssr.sh安装脚本，进行卸载，重新安装一次即可
